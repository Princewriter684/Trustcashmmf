How to Add a WhatsApp Link to Your Website

This page demonstrates a WhatsApp contact button that will open WhatsApp with your number.

Basic Implementation:

Create a link with this format:

<a href="https://wa.me/PHONE_NUMBER">Chat on WhatsApp</a>

Replace PHONE_NUMBER with the full international phone number (without any +, - or spaces).

Example:

<a href="https://wa.me/+254105565102">Chat on WhatsApp</a>

Additional Features:

You can also:

  1. Add predefined text: https://wa.me/PHONE_NUMBER?text=YOUR_MESSAGE
  2. Create a group chat: https://chat.whatsapp.com/GROUP_INVITE_CODE
  3. Include country code but no plus sign (+), spaces, or dashes (-)
